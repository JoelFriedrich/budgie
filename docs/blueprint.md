# **App Name**: Budgie

## Core Features:

- Email Alias Setup: Guides users to set up a unique email alias (e.g., budgetapp+username@gmail.com) for transaction forwarding.
- Email Parsing: Automatically parse incoming emails from the designated alias to extract key transaction details like vendor, amount, date, and description.
- Rule-Based Categorization Tool: AI tool: Match extracted transactions against user-defined rules (e.g., 'Starbucks' to 'Coffee', or regex matches) for automated categorization.
- Transaction Display: Display parsed and categorized transactions in a clear, sortable list within the app.
- Category Management: Allow users to create, edit, and delete custom transaction categories.
- Rule Creation Interface: Provide an intuitive interface for users to define rules for transaction categorization based on vendor, description keywords, or amount patterns.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a modern, serious feel.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop.
- Accent color: Medium gray (#808080) for subtle highlights and important actions.
- Body and headline font: 'Digital', a font resembling a robotic or 7-segment display.
- Use simple, geometric icons in shades of gray to represent transaction categories.
- Emphasize a clean, card-based layout for transaction display, ensuring clarity and ease of interaction.
- Use subtle transitions when categorizing transactions, providing clear feedback to the user.
